## 修改指南

1. 修改本文件中的文字内容
2. 将修改后的内容作为提示词交给 AI
3. AI 会根据此文件同步更新：
   - content/ 下的 JSON 数据文件
   - src/main.jsx 中的页面组件文案
   - 其它需要调整的位置
4. 运行 `npm run build` 构建生产版本
5. 构建输出在 `dist/` 目录，可直接部署

## 管理操作

- 添加文章：编辑 content/knowledge/documents.json
- 添加活动：编辑 content/events/events.json
- 添加视频：编辑 content/media/videos.json
- 添加音乐：将文件放入 music/ 目录，编辑 content/music/tracks.json
- 添加资源：编辑 content/resources/resources.json
- 添加工具：编辑 content/tools/ai-tools.json
- 修改统计：编辑 content/site/stats.json
- 配置评论：编辑 content/forum/giscus.json，填入 GitHub 仓库信息
- 修改项目：编辑 content/projects/projects.json
- 修改权限：编辑 content/members/members.json
