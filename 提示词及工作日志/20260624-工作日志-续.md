### 13. 高端动效系统
- **首屏 Opening Animation**：
  1. 遮罩层向上滑出（`power4.inOut`）
  2. 标题文字从压缩状态解压归位（`expo.out`）
  3. 描述文字与按钮逐项淡入（`power3.out`）
  4. 数据面板从右侧滑入
- **滚动触发动画**：
  - 中文大标题从左侧大幅进场 + clip-path 揭示
  - 卡片依次 stagger 出现 + 3D 旋转恢复
  - 图片 reveal 效果
- **Hero 背景视差**：`scrub: true` 随滚动微移
- 全部使用 GSAP + ScrollTrigger
- 缓动曲线使用 `expo.out` / `power4.out` / `power3.out`
- 无廉价弹跳，节奏缓慢丝滑
- 尊重 `prefers-reduced-motion`

### 14. 四份动效参考落地
- **鼠标动效**（鼠标动效.md）：ClickSpark 组件 — 点击产生金色火花粒子
- **文本动效**（文本动效.md）：GradientText 组件 — CSS 动画渐变文字（"梦游室"主标题）
- **背景动效**（背景动效.md）：PixelSnow 组件 — Canvas 2D 像素漂浮粒子（无 Three.js 依赖）
- **卡片组成部分动效**（卡片组成动效.md）：玻璃拟态卡片 + 3D 透视 hover + 金色光线扫入

### 15. 性能审查与优化
- **JS Bundle**：293 kB（gzip 102 kB）— 仅包含 React + GSAP + lucide-react
- **CSS Bundle**：19 kB（gzip 5 kB）
- **WebGL 移除**：PixelSnow 使用 Canvas 2D 替代 Three.js/WebGL，节省 ~500 kB
- **背景图**：2.4 MB PNG — 使用 CSS background-image 按需加载，不做额外打包
- **ScrollTrigger**：全部设置 `once: true`，避免内存泄漏
- **Scroll 监听**：`passive: true`
- **GSAP context**：路由切换时执行 `ctx.revert()` 清理
- **Canvas 粒子**：使用 IntersectionObserver 仅可见时渲染
- **设备像素比**：Canvas 限制 `Math.min(dpr, 2)` 和 `Math.min(dpr, 1.5)`
- **构建产物**：无重复打包背景图

## 关键修复

1. **Opening Animation 时序问题**：添加 80ms setTimeout 确保 DOM 渲染完成后触发动画
2. **GSAP ctx 作用域 Bug**：将 `ctx` 变量提升到 useEffect 闭包外层，修复清理函数引用错误
3. **音乐播放器状态保持**：MusicPlayer 在 App 根级挂载，Hash 路由切换不卸载
